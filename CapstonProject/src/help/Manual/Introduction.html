Baessagong Action RPG Maker

Version 1.0

 

Instroduction

  Baessagong Action RPG Maker는 프로그래밍을 방법을 숙지하지 못한 일반 사용자도 손쉽게 액션 RPG를 제작할 수 있도록 도와주기 위한 프로그램이다. 시각적으로 모든 제작이 가능하도록 하였기 때문에 프로그래밍에 비해 빠르게 게임을 제작할 수 있다.

  이 문서는 Baessagong Action RPG Maker에 대한 소개와 함께 사용법을 숙지할 수 있도록 제공하는 메뉴얼로, Baessagong Action RPG Maker를 통해 게임을 만들고 하는 사용자에게 빠른 습득을 할 수 있도록 해주기 위한 문서이다.

  스프라이트와 BGM의 저작권 문제로 이 프로그램의 배포는 금지한다.

 

 

Information

1.1 version 1.0의 특징

   - Baessagong Action RPG Maker의 초기 버젼

   - 캐릭터, NPC, Monster 생성 가능. Monster의 기능는 기능이 다소 제한적(행동 타입, 이벤트 생성 등)

   - 무기, 아이템, 스킬은 생성 가능하나 게임 실행기에서 지원하지는 않음.
